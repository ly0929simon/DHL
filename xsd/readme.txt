Note for developer Usage
========================

A) datatypes_global XSD:
========================
1) datatypes_global.xsd = Shipment Validation global schema v1.0, 2.0, 3.0 and 5.2, Global Pickup, Global Routing, Capability and Quote service usage.
2) datatypes_global_v3.xsd = XML Global Mask version 5.22 to version 5.25, XML Schema version 5 Usage only. 
3) datatypes_global_v4.xsd = XML Global Schema version 6.0, XML Schema version 6 Usage only. 
4) datatypes_global_v61.xsd = XML Global Schema version 6.1, XML Schema version 6.1 usage only.
5) datatypes_global_v62.xsd = XML Global Schema version 6.2, XML Schema version 6.2 usage only.

B) datatypes XSD:
=================
1) datatypes.xsd = XML Global Schema version 6.2 datatypes for XML Schema Tracking 6.2 usage only.
